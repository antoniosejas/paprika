Paprika

Javascript interaction framework for paper interfaces

Paper-Based Augmented Reality Games.

Paper interfaces refer to controlling a computer using the manipulation or regular pieces of paper. A camera allows the computer to detect the pieces of paper, and react accordingly. Many well established practices involving paper can thus be reused in interaction design, such as flipping pages, playing with cards, writing on paper, etc.

With the recent development of web technologies, it is now feasible to develop real-time, paper-based applications in the browser. The goal of this project is to explore this new field of interaction. To do so, we intend to develop web-based, augmented reality games using paper interfaces, experimenting various interaction scenarios and gameplay variations. This exploration should result in a software framework for the development of paper-based games, opening the way for the adoption of paper-based augmented reality web applications.
